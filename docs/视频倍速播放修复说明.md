# 视频倍速播放修复说明

## 修改文件清单

| 文件 | 修改说明 |
|------|----------|
| `EvaAnimView.kt` | 修复 `setVideoFps` 方法 |
| `EvaAnimViewV3.kt` | 修复 `setVideoFps` 方法 |
| `EvaAnimConfigManager.kt` | 增加 `isSetFps` 判断，防止 fps 被覆盖 |

#### 位置2：约第192-202行

```kotlin

private fun setNormalMp4(defaultFps: Int) {
    config?.apply {
        isDefaultConfig = true

            playerEva.isNormalMp4 = true

// 修改后
private fun setNormalMp4(defaultFps: Int) {
    config?.apply {
        isDefaultConfig = true
        if (playerEva.videoMode == EvaConstant.VIDEO_MODE_NORMAL_MP4) {
            playerEva.isNormalMp4 = true
        }
        this.defaultVideoMode = playerEva.videoMode
        fps = defaultFps
    }
    // 只有用户没有手动设置 fps 时才使用配置中的 fps
    if (!playerEva.isSetFps) {
        playerEva.fps = config?.fps ?: defaultFps
    }
}
```

## 使用方法

```kotlin
// 在 startPlay 之前调用
animView.setVideoFps(30, 2.0f)   // 2倍速播放
animView.setAudioSpeed(2.0f)     // 音频同步加速
animView.startPlay(file)
```

---

## 问题根因

`EvaAnimConfigManager` 在解析视频配置后，直接用视频原本的 fps 覆盖了用户通过 `setVideoFps` 设置的值，导致倍速失效。
